# Musify Beats

A modern music streaming application built with React, Vite, and Shadcn UI.

## Features

- **Authentication**: User login and registration.
- **Music Player**: Play, pause, and control music playback.
- **Discovery**: Explore trending songs, new releases, and genres.
- **Search**: Find your favorite artists and tracks.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Backend (Auth)**: Spring Boot
- **Backend (Music)**: ASP.NET Core

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Backend Services**:
    Ensure your Spring Boot (port 8080) and ASP.NET Core (port 5000) services are running for full functionality.

## Project Structure

- `src/components`: UI components and layout.
- `src/pages`: Application pages (Home, Search, Auth, etc.).
- `src/services`: API handling for backend services.
- `src/lib`: Utilities and mock data.
